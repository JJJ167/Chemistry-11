The Chemical Elements

Rules for element names and symbols:

  1. One-letter symbol:
    • Needs to be a capital.

    Examples: B, C, O, H, F

  2. Two-letter symbol:
    • First letter is always capitalized.

    Examples: He, Na, Pb, Ag, Au

  • MONATOMIC - Made of 1 atom

    • Examples: He, Ne, Na+, Cl-

  • DIATOMIC - Made of 2 atoms

    • Examples: O2, CO, ClO-

  • TRIATOMIC - Made of 3 atoms

    • Examples: CO2, O3, NOCl

  • POLYATOMIC - Made of many atoms (more than 1)

    • Examples: NO, H2O, NO3-, H2PO4-

Naming Inorganic Compounds

Monatomic Ions

METALS:

  1. Metals with only 1 possible charge
    • Name = Name of Metal + "ion"

      • Examples:

      • Na+ = Sodium ion
      • Ca2+ = Calcium ion

  2. Metals with more than one possible charge (multivelent)
    • Name = Name of Metal + (Charge written using Roman numerals) "ion"

      • Examples:

      • Fe2+ = Iron(II) ion
      • Fe3+ = Iron(III) ion

NON-METALS:

  • Name = Base name + "ide"

    • Examples:

    • Element: Chlorine (Cl)
    • Base name: Chlor-
      • Cl- = Chloride ion

    • Element: Phosphorus (P)
    • Base name: Phosph-
      • P3- = Phosphide ion

Only applies to F, Cl, Br, I, O, S, N, P

Types of Bonds:

  1. Ionic bond:
    • Formed between a metal (can also be ammonium (NH4+)) and a non-metal (can also be a polyatomic).
    • Electrons transfer from one atom to another. Non-metal gains electrons → becomes an anion (-)

  2. Covalent bond:
    • Formed between non-metals.
    • Electrons are shared between atoms.

Writing Formulas

Ionic Compounds Covalent Compounds
  1. Write the symbols of the metal and non-metal (metal is always written first and the non-metal second).
  2. Determine the charges of each ion.
  3. Criss-cross the charges (this is the subscript).
  4. Reduce if possible.

Examples:

  • Sodium chloride → Mg2+Cl- → MgCl2
  • Barium oxide → Ba2+O2- → Ba2O2 → BaO
  1. Write the symbols non-metals.
  2. Use the prefixes to represent the number of each atom (use prefix table below).
  3. Don't balance or criss-cross charges.
  4. Write the least electronegative element first *not important for right now*

Examples:

  • Carbon Dioxide → CO2
  • Dinitrogen tetroxide → N2O4

Naming

Ionic Compounds Covalent Compounds
  1. Write the metal name first.
  2. Write the non-metal name second but change the ending to "ide".
  3. If the metal is a multivalant (ex. Fe, Cu, etc.), write its charge using Roman numerals right after the metal's name.
    • Find the charge by uncrossing the charges. Check if the charges were reduced or not by confirming that the non-metal's charge is correct.

Examples:

  • Li2S → Lithium sulphide
  • FeCl3 → Iron(III) chloride
  • CuO → Cu+ O -
    • Oxygen ion is supposed to be O2-, this means that the charges were reduced.
    • Multiply both charges by 2
    → Cu2+O2- → Copper(II) oxide
  1. Write the name of the first element.
    • If there is more than one atom of the first element, write the correct Greek prefix in front of the name (use prefixes table below).
    • If there is only one atom of the first element, do not use any prefixes.
  2. Write the name of the second element with the correct prefix.
    • Always use a prefix, even if there is only one atom.
  3. Change the ending of the second element to "ide"
  4. Write the least electronegative element first *not important for right now*

Examples:

  • PCl3 → Phosphorus trichloride
  • CO → Carbon monoxide
  • P2S3 → Diphosphorus trisulphide.

Polyatomic Ions

  • Are ions that are made up of two or more atoms covalently bonded.
  • Act as a single charge substance unit.

FORMULAS FOR IONIC COMPOUNDS WITH A POLYATOMIC ION:

  1. Write the metal or a polyatomic cation (ex. ammonium) first.

  2. Write the polyatomic ion second.

  3. Identify the charges for each ion.

  4. Criss-cross the charges (this is the subscript, ignore signs).

  5. Put brackets around the polyatomic ion if there is a charge greater than 1.
    • Write the subscript for the polyatomic after the brackets.

  6. Reduce if possible.

Examples:

  • Sodium nitrate → Na+ and NO3- → NaNO3 - subscript for NO3 is 1, so no brackets are needed.
  • Aluminum sulphate → Al3+ and SO42- → Al2(SO4)3
  • Copper(II) chloride → Cu2+ and ClO3- → Cu(ClO3)2
  • Ammonium sulphate → NH4+ and SO42- → (NH4)2SO4

üíßHydrates

Hydrates are compounds that have water in their formula, however it is not part of the chemical substance. The substance is not "wet".

Naming Formula
  1. Use ionic compound naming rules.
  2. Identify the number of water molecules (hint: look at the subscript after the dot)

    3. Example: MgCl2·6H2O

  3. Use a Greek prefix for the number of water molecules.
  4. Add the word "hydrate"

[Ionic compound name] + [Greek prefix] + "-hydrate"

Examples:

  • MgSO4·9H2O → magnesium sulphate nonahydrate
  • FeSO4·7H2O → Iron(II) sulphate heptahydrate
  1. Use ionic compound formula rules.
  2. Determine the number of water molecules (use the prefix table)
  3. Add a dot (·) and then "nH2O". n = the number of water molecules.

Examples:

  • Calcium carbonate decahydrate → CaCO3·10H2O
  • Copper(II) sulphate pentahydrate → CuSO4·5H2O

üß™Acids

Acids are compounds that make hydrogen ions (H+) in water.

  1. Binary Acids - 2 elements (hydrogen + nonmetal)

    2. Example:

    • HCl
    • HBr

  2. Oxyacids (Ternary Acids) - 3 elements (hydrogen + polyatomic ion)

    3. Example:

    • HNO3
    • H2SO4
Naming Formula

1. Binary Acids

  1. Start with "hydro"- prefix
  2. Write the base of the nonmetal name
  3. Add "-ic acid"

"hydro-" + [nonmetal base] + "-ic acid"

Examples:

  • HCl → hydrochloric acid

  • HBr → hydrobromic acid

1. Binary Acids

  1. Start with "H+".
  2. Write nonmetal symbol.
  3. Criss-cross the charges.

Examples:

  • Hydrofluoric acid → HF
  • Hydrosulphuric acid → H2S

2. Oxyacids

  1. Write the base of the polyatomic ion name
  2. If the polyatomic ends in "-ate", change it to "-ic acid"
  3. If the polyatomic ends in "-ite", change it to "-ous acid"

Examples:

  • H3PO4‚Üí Phosphoric acid

  • H2SO3‚Üí Sulphurous acid

2. Oxyacids

  1. Start with "H+".
  2. Determine the ending:
    • "-ic" will become "-ate"
    • "-ous" will become "-ite"
  3. Find the correct polyatomic ion using it's new ending.
  4. Write the polyatomic ion.
  5. Criss-cross the charges.

Examples:

  • Carbonic acid → polyatomic ion = carbonate → H2CO3
  • Nitrous acid → polyatomic ion = nitrite → HNO2

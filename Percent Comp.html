Percent Composition

Percent composition tells us the percentage by mass that each element makes up in the total mass of a compound.

Percent composition = Total mass of the element in the compound Molar mass of compound x 100%

Example:

  1. Find the percent composition of all the elements in Calcium carbonate.

    2. CaCO3


    • Ca - 40.1 g/mol x 1 = 40.1 g/mol

      • Percent composition = 40.1 g/mol 100.1 g/mol x 100% = 40.1%


    • C - 12.0 g/mol x 1 = 12.0 g/mol

      • Percent composition = 12.0 g/mol 100.1 g/mol x 100% = 12.0%


    • O - 16.0 g/mol x 3 = 48.0 g/mol

      • Percent composition = 48.0 g/mol 100.1 g/mol x 100% = 48.0%

    %Ca = 40.1%

    %C = 12.0%

    %O = 48.0%

Empirical Formula

  • Is the simplest, most reduced whole number ratio of the atoms of each element in a compound.
  • Can be determined using either percent composition or the mass of each element.

    • Example: Glucose is C6H12O6(molecular formula), but it's emperical formula is CH2O

Steps to Calculate:

  1. Find the mass of each element. If you were only given the % composition, change the units from % to g but keep the same value. Write "assume 100g".

  2. Use this formula: Moles = mass (g) atomic mass (g/mol)

  3. Divide all the mole values by the smallest one, this gives you the simplest ratio.

  4. If you get a decimal number, use the table below to see which number you would have to multipy it with to get a whole number.

Prefixes

Example:

  1. A compound contains 40.0% carbon, 6.70% hydrogen, 53.3% oxygen. What is it's empirical formula?

    2. Assume 100g

    • C = 40.0 g 12.0 g/mol = 3.33 mol ÷ 3.33 mol = 1

    • H = 6.70 g 1.0 g/mol = 6.70 mol ÷ 3.33 mol = 2

      • Empirical formula: CH2O

    • O = 53.3 g 16.0 g/mol = 3.33 mol ÷ 3.33 mol = 1


  2. A compound contains 72.4 g Fe, 27.6 g O. What is its empirical formula?
    • Fe = 72.4 g 55.8 g/mol = 1.297 mol ÷ 1.297 mol = 1 x 3 = 3

      • Empirical formula: Fe3O4

    • O = 27.6 g 16.0 g/mol = 1.725 mol ÷ 1.297 mol = 1.33 x 3 = 4


Molecular Formula

  • The actual number of atoms of each element in a compound.
  • Empirical formula that is multiplied by a whole number.

Steps to Calculate:

  1. Find the empirical formula.

  2. Calculate the molar mass of the empirical formula

  3. Use this formula to find N: N = molar mass of molecular formula molar mass of empirical formula

  4. Multiply empirical formula by N.

Example:

  1. A compound contains has an empirical formula CH. The molar mass of this compound is 25.98 g. What is its Molecular formula?
    • mass of empirical formula: 13.0 g/mol
    • N = 25.98 g 13.0 g/mol = 2

    • 2 x (CH) = C2H2

      • Molecular formula:C2H2



Molarity

  • The concentration of a solution (moles of solute dissolved in a solution).
  • Has a symbol of [ ].

Molarity (M) = moles of solute (mol) volume of solution (L)
